1.1 GSEA with Piano

Here we will use piano R package that allows various kinds of gene set enrichment analyses. It also allows various very nice visualization which you can read more about in the package vignette.

1.1.1 Optional: Annotation of DE genes

Since the MSigDB has genes in hgnc symbol format, we can convert our ensemble gene ids to hgcn. In this case we will downlaod some extra id’s fpr potential use.

R 
mart <- useMart("ENSEMBL_MART_ENSEMBL")

Piano package requires a Gene Set and Gene level statistics.

We will use the MSigDB hall mark genes and Kegg pathways for our exploratory analyses.

R 
GSC <- piano::loadGSC("data/MSigDB_files/h.all.v6.2.symbols.gmt.txt", type = "gmt")

GSC_2 <- piano::loadGSC("data/MSigDB_files/c2.cp.kegg.v6.2.symbols.gmt.txt", type = "gmt")

First we prepare the statistics Piano can use for analyses.

R 
dk <- readRDS("data/dg.rds") # This is data that we can use for statistics 

myPval<- dk$pvalue
names(myPval)<-as.character(dk$hgnc_symbol)
myFC<-dk$log2FoldChange
names(myFC)<-as.character(dk$hgnc_symbol)
direction<-ifelse(myFC>0, 1,-1) # The dircetion of regulation is an important distinction in Piano that helps contextualize results.
gene_stats<-cbind(myPval,myFC, direction)

Lets run piano using runGSA function from the piano package. The analyses this package allows takes a bit of time to understand and you are not expected to follow it all in this course. You are welcome to ask questions about the analyses or results if you want to know more about how it works.

R 
piano_gene_sets<-runGSA(geneLevelStats = myPval, directions = direction, gsc=GSC, geneSetStat="fisher", signifMethod="nullDist", nPerm=1000, gsSizeLim=c(1,500))

piano_pathways<-runGSA(geneLevelStats = myPval, directions = direction, gsc=GSC_2, geneSetStat="fisher", signifMethod="nullDist", nPerm=1000, gsSizeLim=c(1,500))
R 
exploreGSAres(piano_pathways)
